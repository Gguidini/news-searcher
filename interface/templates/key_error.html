Erro na Chave da API

O sistema de busca de notícias faz uso da NewsAPI. Para que ela funcione, você precisa de uma chave de acesso válida. Aparentemente sua chave está inválida. O sistema não irá funcionar até que uma chave válida seja colocada no sistema.

Siga os passos abaixo para obter uma nova chave para o sistema.

  1. Faça o Login no site da API

    2. Visite o site da NewsAPI e faça o login com a conta da Sala de Situação. Se você não sabe o login e a senha pergunte ao team leader responsável.

  2. Renovar a chave da API

    3. Clique em "Regenerate Key" e copie a nova chave gerada.

  3. Adicionar a chave no sistema

    4. Cole a nova chave no campo abaixo e clique em Atualizar Chave.

    Depois de atualizar a chave você será redirecionado para as configurações do sistema, mas não precisará mexer em mais nada. Volte para a página de busca e tente novamente.
